Introduction

CronosII Version -0.1.0

CronosII is an Email Client, for the Gnome and GTK environments. Written for users of all levels CronosII is designed to be fast, simply configured, lightweight and functional.

Note: To run CronosII, select Internet CronosII from the Main Menu, or type cronos on the command line.

This Help-Manual ver2.1 describes CronosII Version -0.1.0

Refer to the section called Installing CronosII for Installation instructions.

Refer to the section called Configuration for post install configuration.
