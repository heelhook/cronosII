The Address Book

At this stage both the packages built for CronosII have support for the new address book compiled in. (i.e. the RPM available from the web site and the Debian package available via unstable packages (apt-get install cronosII)

The address book is a relatively new feature for Cronos. As of CronosII version 0.2.2 it works very well however much more is planned for this particular extension to Cronos and so it must still be considered to be an early form of what promises to be a very powerful stand alone application in its own right.
