Terms and Conventions

This manual, like CronosII itself is written for all users, however the new LINUX user has (I Hope) been kept in mind. . . . The advanced user may find little need of a manual for this application and should find greater benefit in reading the source code and notes in the CronosII-0.1.0 tar ball.
